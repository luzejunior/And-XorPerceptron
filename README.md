# And-XorPerceptron

This is a simple implementation of a Perceptron for Binary conditional And, Or, and Xor made using Java Language.

### Instalation:

Clone the repository, go to the folder, and compile with:
```sh
$ javac Main.java
```

Then, run:
```sh
$ java Main
```
